# Automatic_Surat_Pdt
This Repository contains the files made in `Python 3.10`, needed to built a `Surat`. This repo contains a `Online Google Docs` parser by `html`, a `docx editor` methods, and the `json` to built it. Further details is contain within the file

## Usage: ##

Install the required package in the `bash` using
``` bash
pip install -r requirements.txt
```
If you dont have `pip` installed, in the `bash` use
``` bash
sudo apt install python3-pip
```

To run the program, in the `bash` type
``` bash
python main.py
```
and just type the number of `Month` that you want, the `Year` that you need the `Surat`, and the `Number` of the `Surat`.


If used correctly, should looked like this:


![This is a Demo](https://github.com/EndlessWay1/images/blob/main/code_demo_surat_pdt.png?raw=true)


and contains the zip file `Month Year.zip`, in this case `Januari 2025.zip`.


Download the zip file, and you're good to go!
